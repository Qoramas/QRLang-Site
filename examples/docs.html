Colour

Colour is a number from 0-7

0-black 1-blue 2-green 3-cyan 4-red 5-purple 6-yellow 7-white

Docs

All mathmatical methods, brackets, conditionals and loops function the same as javascript

drawbox(x,y,w,h)

Draws a box at position x, y with width w and height h. Colour is set by the previously used colour.

drawbox(x,y,w,h,c)

Draws a box at position x, y with width w and height h. Colour is set by c.

write(string, x, y)

Writes a string starting at position x, y. Colour is set by the previously used colour.

write(string, x, y, c)

Writes a string starting at position x, y. Colour is set by c.
